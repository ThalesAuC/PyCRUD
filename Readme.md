# Sistema C.R.U.D. para teste de conceito:
 - criado em Python, implementado mongoDB para o banco de dados, implementação em docker para deploy rapido.

 1- Para rodar o programa use 'git clone https://github.com/ThalesAuC/PyCRUD'
 2- depois para iniciar o banco de dados use: 'docker-compose up -d mongodb'
 3- e por fim use 'docker-compose up --build smartficha_0.1'
